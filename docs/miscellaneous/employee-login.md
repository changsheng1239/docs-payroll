### How to Login as Employee

1. In SQL Payroll, go to **Human Resource** -> **Maintain Employee**

2. Click on **Change Password**

   ![1](/img/employee-login/1.png)

3. Enter a new password for the employee

   ![2](/img/employee-login/2.png)
